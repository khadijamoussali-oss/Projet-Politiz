# Projet-Politiz
L’idée est de créer une solution ludique, neutre, avec une progression qui permette d'apprendre les bases de la politique française en quelques minutes par jour. 
Ce sera en mobile first c’est-à-dire ciblé pour une interface graphique web mobile mais adapté aux ordinateurs. 
On s’inspire ici d’applications de micro-learning type Duolingo.
